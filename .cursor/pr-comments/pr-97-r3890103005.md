## 🟠 Major — Novo (1 actionable)

### 1. `inlineComments` retorna comentários de todos os usuários, não só CodeRabbit

**Arquivo**: [.claude/skills/pr-comments/SKILL.md:46-52](.claude/skills/pr-comments/SKILL.md#L46)
**Tipo**: Potential issue

No modo "buscar todos os comentários do CodeRabbit", a query não filtra por `user.login`. Retorna comentários de qualquer revisor, não só o CodeRabbit. Além disso, o `reviewBody` pode emitir múltiplos valores se houver mais de um review do CodeRabbit.

**Fix**:
```diff
 inlineComments=$(gh api repos/$repo/pulls/$prNum/comments \
-  --jq '[.[] | {path: .path, line: (...), body: .body, user: .user.login}]')
+  --jq '[.[] | select(.user.login | contains("coderabbit")) | {path: .path, line: (...), body: .body, user: .user.login}]')

 reviewBody=$(gh api repos/$repo/pulls/$prNum/reviews \
-  --jq '.[] | select(.user.login | contains("coderabbit")) | .body')
+  --jq '[.[] | select(.user.login | contains("coderabbit")) | .body] | join("\n\n")')
```
